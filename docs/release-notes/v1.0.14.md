# ConfigPilot v1.0.14

本版本校正 GPT-5.6 的普通思考等级，并让 ConfigPilot 优先从远端模型目录同步模型支持的 Effort，减少客户端静态配置与上游能力不一致的问题。

## 主要更新

- GPT-5.6 Sol 的普通思考等级统一为“轻度 / 中 / 高 / 极高”四档，对应 `low / medium / high / xhigh`。
- 获取模型时优先读取上游 `/v1/models` 返回的 `supported_reasoning_levels`。
- 上游模型接口未提供 Effort 时，通过 `codex debug models` 刷新并读取 Codex 官方远端模型目录。
- `Max` 与 `Ultra` 属于需要单独开启的特殊模式，不再混入普通思考等级选择器。
- 两种远端目录均不可用时，继续使用经过校正的四档内置回退，不影响离线配置。
- 远端目录更新后，当前页面和思考等级下拉列表会即时刷新。

## 升级说明

ConfigPilot 已包含应用内更新器。Windows 用户可在应用内完成下载、静默安装和重启；macOS 用户检测到新版后会打开官方 Release 页面。

升级后重新打开 Codex 配置页即可触发远端 Effort 刷新；点击“获取模型”也会同步当前中转模型接口提供的能力信息。

## macOS 首次安装须知

当前 macOS 版本仅支持 Apple Silicon（M 系列芯片），不支持 Intel Mac。

维护者目前没有 Apple Developer Program 会员，因此 DMG 尚未使用 Developer ID 签名和 Apple 公证。macOS Gatekeeper 可能显示“ConfigPilot 已损坏，无法打开”，但这不代表 DMG 文件真的损坏。

请先把 `ConfigPilot` 拖入 `Applications`，再打开“终端”执行：

```bash
xattr -dr com.apple.quarantine "/Applications/ConfigPilot.app"
open "/Applications/ConfigPilot.app"
```

同一已安装版本通常只需执行一次。请仅从本项目官方 Releases 下载。
