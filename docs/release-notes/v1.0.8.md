# ConfigPilot v1.0.8

> Windows 用户请优先升级到 [v1.0.9](https://github.com/aki-riko/ConfigPilot/releases/tag/v1.0.9)。v1.0.9 修复了旧版升级后安装目录仍为 `CodexConfig` 和任务栏显示通用图标的问题。

本版本完成产品更名、GPT-5.6 配置支持和配置页体验重构。

## 主要更新

- 产品正式更名为 **ConfigPilot**。
- GPT-5.6 支持六档中文思考等级，选择模型时默认使用最高档 `max`。
- GPT-5.5 与其他模型默认使用最高可用档 `xhigh`。
- GPT-5.5 使用 `258400 / 245000 / 6000` 上下文预设；GPT-5.6 使用 `372000 / 353000 / 6000`。
- 配置页重构为连接、模型、上下文和兼容性分区，窄窗口自动切换单列。
- 新增固定操作栏和“已同步 / 待应用”状态提示。
- PrismQML 升级到 `0.2.24.9`。

## macOS 首次安装须知

当前 macOS 版本仅支持 Apple Silicon（M 系列芯片），不支持 Intel Mac。

维护者目前没有 Apple Developer Program 会员，因此 DMG 尚未使用 Developer ID 签名和 Apple 公证。macOS Gatekeeper 可能显示“ConfigPilot 已损坏，无法打开”，但这不代表 DMG 文件真的损坏。

请先把 `ConfigPilot` 拖入 `Applications`，再打开“终端”执行：

```bash
xattr -dr com.apple.quarantine "/Applications/ConfigPilot.app"
open "/Applications/ConfigPilot.app"
```

同一已安装版本通常只需执行一次。请仅从本项目官方 Releases 下载。
