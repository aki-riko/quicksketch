## macOS 安装须知

- macOS 产物 `CodexConfig_1.0.7_arm64.dmg` 仅支持 Apple Silicon，不支持 Intel Mac。
- 维护者目前没有 Apple Developer Program 会员，因此该版本未使用 Developer ID 签名和 Apple 公证。
- macOS Gatekeeper 可能显示“CodexConfig 已损坏，无法打开”，但 Release 文件已核对 SHA-256，问题不是 GitHub 上传损坏。

请先把 `CodexConfig` 拖入 `Applications`，然后打开“终端”执行：

```bash
xattr -dr com.apple.quarantine "/Applications/CodexConfig.app"
open "/Applications/CodexConfig.app"
```
